# WordFrequencyCounter

This project counts word frequencies in a text using Python.

## Example

Input:

```text
The quick brown fox jumps over the lazy dog. The fox is quick.
```

Output:

```python
{
'the': 3,
'quick': 2,
'fox': 2,
'brown': 1,
'jumps': 1,
'over': 1,
'lazy': 1,
'dog': 1,
'is': 1
}
```

## Technologies Used

- Python
- Dictionaries
- Loops
- Functions
